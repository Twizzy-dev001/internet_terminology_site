Hypertext Transfer Protocol (HTTP)

Hypertext Transfer Protocol (HTTP) is the communication protocol used to transfer web pages from a server to a browser. When you enter a URL in your browser, HTTP sends a request to the server, and the server responds by sending back the requested webpage. It is the foundation of data communication on the World Wide Web.
